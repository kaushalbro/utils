Create SQL back -> sql_dbbackup_script.sh

  3-> sudo nano ./sql_dbbackup_script.sh

  4-> sudo chmod +x  ./sql_dbbackup_script.sh
  5-> ./sql_dbbackup_script.sh   (to run)
